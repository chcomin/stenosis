# Example scripts

This folder contain example scripts for running stenosis diagnoses. Note that the package must be installed for the scripts to work. See the main readme for instructions. You also need to install additional packages to run the scripts:

```bash
conda install segmentation-models-pytorch trl transformers accelerate timm peft bitsandbytes notebook matplotlib
```

The [torchtrainer](https://github.com/chcomin/torchtrainer) package.
